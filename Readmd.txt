SDK Version : Android 11(R)
구현된 기능
1. 첫번째 화면
	1) 앱 접속 페이지, 회원 ID/PASSWORD, 로그인 버튼 구현
	2) 로그인 버튼 클릭 시 기존 가입한 회원 ID, 비밀번호 체크 후 TOAST 메세지 출력
	3) 회원 가입 메세지 없어도 메인버튼을 통해 상품출력으로 이동 가능

2. 두번째 화면
	1) 회원가입 페이지 구현
	2) ID 중복 검사 구현, 패스워드 입력시 정규식을 활용해 정해진 조건이 아니면 오류 메세지 나오도록 설정
	3) 회원 정보를 프레퍼렌스로 저장

3. 세번째 화면
	1) 상품명 , 상품이미지를 GridView를 통해 처음에 5개를 보여줌
	2) 이 후에 이름을 텍스트 입력창에 입력하고 삭제를 누르면 상품리스트에서 상품 물건이 있는지 체크하여 삭제
	3) 추가의 경우에는 텍스트 입력창에 텍스트를 입력과 갤러리에서 선택 과정 두 과정이 있으면 상품 올리기를 통해 업로드 가능 단 사진이 없다면 오류 메세지 출력
	4) 회원 정보를 누르면 가입한 정보를 프레퍼런스를 통해 가져와 보여주고 아닐 경우에는 다이알로그가 화면에 나와 가입할건지 안할건지 물어보고 대답에 따라 가입시 두번째로 이동한다.
